This website is the result of my final project on the classification of herbal and poisonous leaves using various transfer learning models combined with rule-based system techniques to provide further information on the classified leaves. You can see my journal [here](https://doi.org/10.36341/rabit.v10i2.6025).

## DATASET
The dataset used in this study can be downloaded [here](https://drive.google.com/file/d/12mxVGgAWwoAal-xISKWHf54FT7dIMHxL/view?usp=sharing)
<br> The dataset is sourced from Kaggle: [dataset daun](https://www.kaggle.com/datasets/cholisah/datasetdaun) dan [Dataset Tanaman Herbal](https://www.kaggle.com/datasets/anefiamutiaraatha/dataset-tanaman-herbal).
